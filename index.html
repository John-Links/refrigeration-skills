🛠️ 7 个 Skills 一览

1
物性查询

coolprop_query

查任意制冷剂的饱和温度/饱和压力/任意工况物性。底层用 CoolProp 7.0+，NIST 标准。

2
图数据生成

coolprop_ph

一键生成 p-h 图（压焓图）所有数据点。饱和液线、饱和蒸汽线、过热区等压线全包。

3
表生成

coolprop_sat_table

生成完整饱和性质表（温度→压力、密度、焓、熵）。支持 ASCII 表格输出，方便贴报告。

4
单位换算

unit_converter

制冷常用单位换算 13 类：温度、压力、能量、功率、流量、管径、质量、长度等。

5
湿空气

psychrometric

焓湿图 6 参数互算：干球、湿球、露点、相对湿度、含湿量、焓值。HVAC 设计必备。

6
水力计算

pipe_pressure_drop

管道沿程压降 Darcy-Weisbach 计算。支持水、乙二醇溶液、雷诺数自动判定流型。

7
图数据生成

coolprop_t_s

一键生成 T-s 图（温熵图）所有数据点。饱和曲线、等温线、等压线全包，看效率。

🚀 3 步上手

第 1 步：克隆仓库

git clone https://github.com/John-Links/refrigeration-skills.git

第 2 步：安装依赖

pip install -r requirements.txt

第 3 步：开始使用

from skills import query_saturation_by_temperature
result = query_saturation_by_temperature("R134a", 25, "C")
print(result["saturation"]["P_sat_kpa"]) # 665.38 kPa
